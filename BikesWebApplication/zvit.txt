Мета: Покращити знання у роботі з базами данних.

Я виконав даний проект на мові С# за допомогою ASP.NET Core та Entity Framework Core,
де використовується БД та SQL-запити для отримання інформації.
На виході маю сайт-базу даних, де можу працювати з даною БД.

За предметну область обрано базу данних велосипедів, їх фабрик, магазинів, міст, де знаходяться 
фабрики та магазини, також розміри та кольори велосипедів.

При вході на веб-сторінку користувач має увійти або зареєструватись на ресурсі. Для цього під капотом
я створив ще одну базу данних спеціально для роботи з користувачами.

Далі користувач має право передивитись вміст ресурсу, а отже таблиць бази данних.
Деякі співвідношення між данними таблиць додотково представлено у вигляді діаграм для кращого
візуального сприйняття.

Є можливість додавати, видаляти, редагувати данні цих таблиць, а також переглядати детальні дані,
що відображають додаткові відомості з інших таблиць.

Проведено роботу з front-end для кращого візуального сприйняття.

Висновок: Опанував роботу з базами данних за допомогою Microsoft SQL Server Manager, ASP.NET Core,
Entity Framework Core. Детальніше поцікавився та зрозумів матеріал з приводу
БД та SQL-запитів для отримання інформації.
